# Legacy Logjamming Prototype

The previous one-file prototype has been preserved under `legacy/logjamming/`.

It contains:

- the old fullscreen review UI;
- local `like`/`rating` decisions through `localStorage`;
- R2 listing through a Pages Function;
- the seed `films.json` and `images.json` files.

This code is no longer part of the public build. It remains useful as a reference for future private backstage tooling: selection, notes, ratings, and album review flows.
